Master Theorem
Merge Sort    T(n) = 2T(n/2) + O(n)    ------>  O(n logn)

Quick Sort    T(n) = 2T(n/2) + O(n)    ------>  O(n logn)

Bubble Sort             O(n²)
Insertion Sort          O(n²)
Selection Sort          O(n²)

Linear Search           O(n)
Binary Search           O(logn)
